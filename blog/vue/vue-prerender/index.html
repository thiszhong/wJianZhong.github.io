Vue prerender 预渲染

直接看项目：https://github.com/thiszhong/vue-prerender

由 Vue cli 搭建，prerender-spa-plugin 进行预渲染，vue-meta 管理/自定义 head 信息 的简单项目模板。

适用于需要 SEO 的站点、活动单页、散页。

可以分项目打包输出，自动生成对应 prerender 路由。
